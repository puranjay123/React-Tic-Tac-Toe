## Project Description
This is a basic React app to play the game of TicTacToe. The project is similar to the one given on the React Documentation but has been recreated after learning React.

**The react app can do the following:**
1. Tell who's turn it is
2. Change what is written in each button on the tic-tac-toe board
3. Declare a winner

The project has the following component structure:
* Game
    * Board
        * Square
        * Square
        * Square
        * Square
        * Square
        * Square
        * Square
        * Square
        * Square
